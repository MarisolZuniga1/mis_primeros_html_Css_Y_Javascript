Aprendiendo HTML

HTML es un lenguaje con el que se define el contenido de las paginas web

Intructor: Kiko Palomares

Comandos mas utilizados:

"Todos los comandos seran escritos dentro de < >"

Comandos Funcionalidad
hr Sirve para dibujar una linia al ancho de la pagina, para marcar las diferentes secciones.
pre El texto se escribe tal cual fue en el codigo de programación.
b Escribe el texto en negritas.
p Nos sirve para escribir un parrafo en la pagina web.
strong El texto se escribe en negrita, pero se resalta como una palabra importante.
i Escribe el texto en cursiva.
em El texto se escribe en cursiva, pero se resalta como una palabra importante.
small El texto se escribe mas pequeño.
mark Resalta el texto como un marcatexto.
del La palabra se escribe tachada.
ins El texto se muestra subrayado.
sub Todo escrito dentro de esta etiqueta aparece como un subindice.
sup Todo lo escrito dentro de esta etiqueta aparece coo un superindice.
d La palabra se escribe tachada.
!-- comentario -- Sirve para escribir comentarios en el codigo.
